### 1. Основные понятия кинематики

-   **Система отсчета:** совокупность тела отсчета, связанной с ним системы координат и прибора для измерения времени.
    
-   **Траектория:** воображаемая линия, вдоль которой движется тело.
    
-   **Путь ($s$):** длина участка траектории, пройденного телом за данный промежуток времени (скаляр).
    
-   **Радиус-вектор ($\vec{r}$):** вектор, соединяющий начало координат с положением точки в данный момент времени.
    
-   **Перемещение ($\Delta \vec{r}$):** вектор, соединяющий начальное положение точки с её последующим положением.
    
-   **Скорость ($\vec{v}$):** векторная величина, характеризующая быстроту перемещения.
    
    $$\vec{v} = \frac{d\vec{r}}{dt}$$
    
-   **Ускорение ($\vec{a}$):** векторная величина, характеризующая быстроту изменения скорости.
    
    $$\vec{a} = \frac{d\vec{v}}{dt} = \frac{d^2\vec{r}}{dt^2}$$
    

----------

### 2. Кинематические характеристики вращательного движения

-   **Угловое перемещение ($Δφ$):** вектор, модуль которого равен углу поворота, а направление определяется по правилу правого винта. 
    
-   **Угловая скорость ($\vec{\omega}$):** векторная величина, определяющая быстроту вращения.
    
    $$\vec{\omega} = \frac{d\vec{\varphi}}{dt}$$
    
-   **Угловое ускорение ($\vec{\varepsilon}$):** характеризует быстроту изменения угловой скорости.
    
    $$\vec{\varepsilon} = \frac{d\vec{\omega}}{dt}$$
    

**Связь линейных и угловых величин:**

-   Путь и угол: $s = \varphi R$
    
-   Линейная скорость: $v = \omega R$
    
-   Тангенциальное ускорение: $a_\tau = \varepsilon R$
    
-   Нормальное ускорение: $a_n = \omega^2 R$
    

----------

### 3. Ускорение при криволинейном движении

Полное ускорение $\vec{a}$ раскладывается на две составляющие:

1.  **Тангенциальное ($a_\tau$):** направлено по касательной, отвечает за изменение _модуля_ скорости.
    
    $$a_\tau = \frac{dv}{dt}$$
    
2.  **Нормальное ($a_n$):** направлено к центру кривизны, отвечает за изменение _направления_ скорости.
    
    $$a_n = \frac{v^2}{R}$$
    
    **Полное ускорение:** $a = \sqrt{a_\tau^2 + a_n^2}$
    

----------

### 4. Основные понятия динамики

-   **Сила ($\vec{F}$):** мера механического воздействия на тело со стороны других тел или полей.
    
-   **Масса ($m$):** мера инертности тела при поступательном движении.
    
-   **Импульс ($\vec{p}$):** количество движения.
    
    $$\vec{p} = m\vec{v}$$
    
-   **Импульс силы:** характеризует действие силы в течение определённого промежутка времени. 
$$\vec{F} dt = d\vec{p}$$ (изменение импульса равно импульсу силы).
    

**Законы Ньютона:**

1.  **I закон:** существуют системы отсчета (инерциальные), в которых тело сохраняет состояние покоя или равномерного прямолинейного движения, пока на него не действуют силы.

    V=const, при F=0
   
3.  **II закон:** ускорение тела прямо пропорционально равнодействующей всех сил, приложенных к телу, и обратно пропорционально его массе. 

     $$\vec{F} = m\vec{a}$$ или в общем виде $$\vec{F} = \frac{d\vec{p}}{dt}$$
    
4.  **III закон:** тела взаимодействуют силами, равными по модулю и противоположными по направлению.
    
    $$\vec{F}_{12} = -\vec{F}_{21}$$
    

----------

### 5. Система тел. Центр масс

-   **Центр масс ($\vec{r}_c$):** точка, характеризующая распределение масс в системе.
    
    $$\vec{r}_c = \frac{\sum m_i \vec{r}_i}{\sum m_i}$$
    
-   **Закон движения центра масс:** центр масс системы движется так, как двигалась бы материальная точка с массой, равной массе всей системы, под действием суммы внешних сил.
    
    $$M \vec{a}_c = \sum{\vec{F}_{внеш}}$$
    
-   **Закон сохранения импульса:** если сумма внешних сил равна нулю ($\sum \vec{F}_{внеш} = 0$), то импульс системы сохраняется.
    
    $$\vec{p} = \text{const}$$
    

----------

### 6. Динамика вращательного движения

-   **Момент силы ($\vec{M}$):** векторная величина, которая характеризует вращательное действие силы на тело относительно определённой точки или оси.
$\vec{M} = [\vec{r} \times \vec{F}]$. Модуль $M = F \cdot l$, где $l$ — плечо силы.
    
-   **Момент инерции ($I$):** мера инертности при вращении. Для материальной точки $I = mr^2$. Для твердого тела:
    
    $$I = \int r^2 dm$$
    
-   **Момент импульса ($\vec{L}$):** физическая величина, характеризующая количество вращательного движения.
Для вращающегося тела $\vec{L} = I\vec{\omega}$.
    
-   **Теорема Штейнера:** момент инерции относительно произвольной оси равен сумме момента инерции относительно параллельной оси, проходящей через центр масс ($I_c$), и произведения массы тела на квадрат расстояния между осями ($d$).
    
    $$I = I_c + md^2$$
    

----------

### 7. Момент импульса и основной закон динамики

-   **Закон сохранения момента импульса:** если суммарный момент внешних сил равен нулю ($\sum \vec{M}_{внеш} = 0$), то момент импульса системы сохраняется.
    
    $$\vec{L} = \text{const}$$
    
-   **Основной закон динамики вращательного движения:** скорость изменения момента импульса равна суммарному моменту внешних сил.
    
    $$\frac{d\vec{L}}{dt} = \vec{M} \quad \text{или} \quad I\varepsilon = M$$
    

----------

### 8. Работа и мощность

-   **Работа постоянной силы:** результат действия силы на тело при его перемещении. $$A = \vec{F} \cdot \vec{s} = Fs \cos \alpha$$
    
-  **Работа переменной силы:** действие силы при перемещении тела, если величина или направление силы изменяются в процессе движения. $$A = \int \vec{F} d\vec{r}$$
    
-   **Работа при вращении:** результат действия силы при вращательном движении твёрдого тела вокруг неподвижной оси. $$A = \int M d\varphi$$
    
-   **Мощность ($P$):** быстрота совершения работы.
    
    $$P = \frac{dA}{dt} = \vec{F} \cdot \vec{v}$$
    

----------

### 9. Кинетическая энергия

-   **Для поступательного движения:** $E_k = \frac{mv^2}{2}$
    
-   **Для вращательного движения:** $E_k = \frac{I\omega^2}{2}$
    
-   **Теорема о кинетической энергии:** изменение кинетической энергии тела равно работе всех сил, действующих на него.
    
    $$\Delta E_k = A$$
    
-   **Кинетическая энергия системы тел:** сумма кинетических энергий всех точек системы.
    

----------

### 10. Консервативные силы и потенциальная энергия

-   **Консервативные силы:** силы, работа которых не зависит от формы пути, а только от начального и конечного положений (например, сила тяжести, сила упругости).
    
-   **Потенциальная энергия ($E_p$):** энергия, зависящая от взаимного расположения тел или частей тела. Работа консервативных сил равна убыли потенциальной энергии:
    
    $$A = -\Delta E_p$$
    
-   **Связь силы и энергии:** $\vec{F} = -\text{grad } E_p$ (сила направлена в сторону наиболее быстрого убывания энергии).
    

----------

### 11. Полная механическая энергия

-   **Полная энергия:** она характеризует способность системы совершать механическую работу и зависит от движения тел и их положения в пространстве.  
$$E = E_k + E_p$$
    
-   **Закон сохранения энергии:** в замкнутой системе, где действуют только консервативные силы, полная механическая энергия остается постоянной.
    
    $$E_k + E_p = \text{const}$$
    
    Если действуют неконсервативные силы (например, трение), то $\Delta E = A_{fric}$.
    ----------

## 12. Постулаты теории относительности (СТО)

Специальная теория относительности Эйнштейна базируется на двух постулатах:

1.  Принцип относительности: Все физические законы одинаковы во всех инерциальных системах отсчета (ИСО).
    
2.  Постулат постоянства скорости света: Скорость света в вакууме $c$ одинакова во всех ИСО и не зависит от скорости движения источника или наблюдателя ($c \approx 3 \cdot 10^8$ м/с).
    

Следствия:

-   Относительность одновременности: События, одновременные в одной ИСО, могут не быть одновременными в другой, движущейся относительно первой.
    
-   Замедление времени: Время в движущейся системе течет медленнее:
    
    $$\Delta t = \frac{\Delta t_0}{\sqrt{1 - \beta^2}}, \text{ где } \beta = \frac{v}{c}$$
    
-   Сокращение длины: Длина предмета в направлении движения уменьшается:
    
    $$L = L_0 \sqrt{1 - \beta^2}$$
    

----------

## 13. Связь массы, энергии и импульса

-   Связь массы и энергии: $E_0 = m c^2$.
    
-   Полная энергия: $E = mc^2 = \frac{m_0 c^2}{\sqrt{1 - \beta^2}}$.
    
-   Связь энергии и импульса:
    
    $$E^2 = (pc)^2 + (m_0 c^2)^2$$
    
    

----------

## 14. Электрический заряд и закон Кулона

-   Закон кратности (квантования): Любой заряд кратен элементарному заряду $e \approx 1,6 \cdot 10^{-19}$ Кл
    
-   Закон сохранения заряда: В изолированной системе алгебраическая сумма зарядов постоянна: $\sum q_i = \text{const}$.
    
-   Закон Кулона: Сила взаимодействия двух точечных зарядов в вакууме:
    
    $$F = k \frac{|q_1 q_2|}{r^2} = \frac{1}{4\pi\varepsilon_0} \frac{|q_1 q_2|}{r^2}$$
    
-   Напряженность поля ($\vec{E}$): Силовая характеристика поля. $\vec{E} = \frac{\vec{F}}{q}$.

- Силовые линии — линии, касательные к которым в каждой точке совпадают с направлением $\vec{E}$.
    
-   Принцип суперпозиции: Напряженность поля системы зарядов равна векторной сумме напряженностей от каждого заряда: $\vec{E} = \sum \vec{E}_i$.
    

----------

## 15. Потенциал электрического поля

-   Потенциал ($\phi$): Энергетическая характеристика поля. $\phi = \frac{W_p}{q}$.
    
-   Разность потенциалов (напряжение): $U = \phi_1 - \phi_2 = \frac{A}{q}$.
    
-   Связь $E$ и $\phi$: Напряженность равна градиенту потенциала со знаком минус:
    
    $$\vec{E} = -\text{grad } \phi \text{ (в одномерном случае: } E = -\frac{d\phi}{dx})$$
    

----------

## 16. Теорема Гаусса
Поток вектора напряженности - это скалярная величина, характеризующая «количество» силовых линий поля, проходящих через заданную поверхность. 

Поток вектора напряженности $\Phi_E$ через замкнутую поверхность в вакууме пропорционален суммарному заряду внутри неё:

$$\Phi_E = \oint_S \vec{E} \cdot d\vec{S} = \frac{Q_{внутр}}{\varepsilon_0} $$

Примеры расчета $E$:

-   Точечный заряд: $E = \frac{q}{4\pi\varepsilon_0 r^2}$.
    
-   Бесконечная плоскость: $E = \frac{\sigma}{2\varepsilon_0}$.
    
-   Сфера (снаружи): $E = \frac{q}{4\pi\varepsilon_0 r^2}$ (как точечный заряд).
    

----------

## 17. Электроемкость и конденсаторы

Электроемкость: $C = \frac{q}{U}$. Единица измерения — Фарад (Ф).

Емкость конденсаторов:

-   Плоский: $C = \frac{\varepsilon \varepsilon_0 S}{d}$.
    
-   Сферический: $C = 4\pi\varepsilon\varepsilon_0 \frac{R_1 R_2}{R_2 - R_1}$.
    
-   Цилиндрический: $C = \frac{2\pi\varepsilon\varepsilon_0 L}{\ln(R_2/R_1)}$.
    

Соединения:

-   Параллельное: $C_{total} = C_1 + C_2 + \dots$
    
-   Последовательное: $\frac{1}{C_{total}} = \frac{1}{C_1} + \frac{1}{C_2} + \dots$
    

----------

## 18. Энергия электрического поля

-   Энергия системы зарядов: $W = \frac{1}{2} \sum q_i \phi_i$.
    
-   Энергия заряженного конденсатора:
    
    $$W = \frac{qU}{2} = \frac{CU^2}{2} = \frac{q^2}{2C}$$
    
-   Объемная плотность энергии: Энергия, приходящаяся на единицу объема поля:
    
    $$w = \frac{W}{V} = \frac{\varepsilon\varepsilon_0 E^2}{2}$$
    

----------

## 19. Электрический ток

Условия протекания: наличие
свободных носителей заряда и наличие электрического поля (разности потенциалов).

-   Сила тока: $I = \frac{dq}{dt}$ (скорость переноса заряда).
    
-   Плотность тока: $j = \frac{I}{S} = n e v_d$, где $v_d$ — дрейфовая скорость.
    

----------

## 20. ЭДС и сопротивление

-   Сторонние силы: Силы неэлектростатического происхождения (химические, магнитные и т.д.), поддерживающие ток.
    
-   ЭДС ($\mathcal{E}$): Работа сторонних сил по перемещению единичного заряда: $\mathcal{E} = \frac{A_{st}}{q}$.
    
-   Напряжение ($U$): Учитывает работу и кулоновских, и сторонних сил.
    
-   Сопротивление проводника: $R = \rho \frac{l}{S}$, где $\rho$ — удельное сопротивление.
    

----------

## 21. Закон Ома и правила Кирхгофа

4 вида закона Ома:

1.  Для участка цепи: $I = \frac{U}{R}$.
    
2.  Для полной (замкнутой) цепи: $I = \frac{\mathcal{E}}{R + r}$ (где $r$ — внутреннее сопротивление).
    
3.  В дифференциальной (локальной) форме: $\vec{j} = \sigma \vec{E}$ (где $\sigma = 1/\rho$ — проводимость).
    
4.  Для неоднородного участка цепи: $I = \frac{\phi_1 - \phi_2 + \mathcal{E}}{R}$.
    

-   Закон Джоуля — Ленца: Количество теплоты $Q = I^2 R t$.
    

Правила Кирхгофа:

1.  Узлов: Сумма токов в узле равна нулю: $\sum I_k = 0$.
    
2.  Контуров: Алгебраическая сумма падений напряжения в замкнутом контуре равна сумме ЭДС в нем: $\sum I_k R_k = \sum \mathcal{E}_k$.
